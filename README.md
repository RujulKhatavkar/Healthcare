# Saathi App - Vite + React + Tailwind (Demo)

This is a modular, runnable demo of the Saathi app tailored for elderly users with accessibility priorities.

## Quick start (in VS Code)
1. Open this folder in VS Code.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start Vite dev server.
4. Open http://localhost:5173 in your browser.

Notes:
- The project uses `lucide-react` for icons.
- Camera / OCR / contact access are mocked. Production integration requires native/browser permissions & backend.
